### 문제 요약 : "one4seveneight" → 1478
### 해시맵, 문자열(replace)

```python
def solution(s):
    answer = 0
    
    num_alpha_set = {"zero":"0","one":"1","two":"2","three":"3","four":"4"
                     ,"five":"5","six":"6", "seven":"7", "eight":"8", "nine":"9"}
    
    for nas in num_alpha_set:
        s = s.replace(nas,num_alpha_set[nas])
    
    answer = int(s)
    
    return answer
```
